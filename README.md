# VacoTech Consulting Website

A premium, high-performance static website for VacoTech Consulting Inc.

## Features
- **Modern Design**: Dark mode with glassmorphism and smooth entrance animations.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **Zero Backend**: Built with pure HTML5, CSS3, and Vanilla JavaScript. No server-side runtime required.
- **SEO Optimized**: Semantic HTML and descriptive meta tags.

## Hosting on cPanel / Namecheap
1. Log in to your **cPanel**.
2. Go to **File Manager**.
3. Navigate to the `public_html` directory.
4. Upload all the files and folders in this project:
   - `index.html`
   - `about.html`
   - `solutions.html`
   - `case-studies.html`
   - `careers.html`
   - `contact.html`
   - `css/` (folder)
   - `js/` (folder)
   - `assets/` (folder)
5. Your website will be live at your domain!

## Technologies Used
- **HTML5**: Structure and semantics.
- **CSS3**: Custom design tokens, CSS Grid, Flexbox, and Glassmorphism.
- **Vanilla JS**: Interactive elements and scroll animations.
- **Lucide Icons**: Modern icon library.
- **Google Fonts**: Outfit (Headings) and Inter (Body).
